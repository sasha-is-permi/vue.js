Installation
Vue не поддерживает IE8 и ниже, потому что он использует функции
 ECMAScript 5, которые нельзя изменить в IE8. Однако он поддерживает
 все совместимые с ECMAScript 5 браузеры 

1) Можно скачать как версию для разработки (если надо что-то изменить 
в vue.js):
Development version,
Так и версию готовую: production version

2) Так же vue.js можно подключить через cdn:
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
(если подключить в head- быстрее может идти загрузка)

3) // Можно установить vue.js через npm:

npm init
npm install vue -g

в web storm #app <tab> - создается <div id="app">
