# project1
Пробный проект на vue и firebase:
Открытая система хранения фотографий.
Можно зарегистрироваться в системе, залогиниться. 
На главной странице- просмотр и своих и чужих фотографий.
В меню My Ads- просмотр только своих фотографий с возможностью
их редактирования
